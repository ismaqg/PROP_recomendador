Recomendador de items utilizando los algoritmos K-means y SlopeOne. Dentro de la carpeta DOCS/Documentacio puede encontrarse el manual de uso, el pseudocódigo de los algoritmos usados, etc. Mi rol en este trabajo, entre otros, fue el de desarrollar estos dos algortimos para la recomendación. La calificación final del proyecto fue de Matrícula de Honor.


Propyecto_src contiene el código fuente y en DOCS están todos los documentos que dan información sobre el programa. El manual de usuario muestra las vistas de la aplicación, por lo que recomiendo visitarlo.


Este NO es el original que contiene todo lo necesario para compilar y ejecutarse (makefiles, juegos de prueba, bases de datos, etc.), sino una versión con la documentación y los códigos para ver simplemente estos aspectos.
