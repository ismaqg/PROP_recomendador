\doxysubsection{Detailed Description}
Clase que contiene los datos de los cálculos del algoritmo K\+Means. 

Ofrece operaciones para que el Data\+Manager pueda acceder a información precargada del algoritmo K\+Means y para poder guardar una nueva configuración de éste.
