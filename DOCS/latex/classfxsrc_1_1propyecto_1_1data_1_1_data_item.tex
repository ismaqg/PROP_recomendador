\doxysubsection{Detailed Description}
Clase que contiene los datos de todos los Item del sistema. 

Ofrece operaciones para que el Data\+Manager pueda añadir Items nuevos a partir de un String que sería la representación del Item a añadir en formato C\+SV. También ofrece operaciones para que el Data\+Manager pida todos los Items, un Item en concreto a partir de la posición del Array (útil para bucles) o incluso un Item a partir de su ID.
