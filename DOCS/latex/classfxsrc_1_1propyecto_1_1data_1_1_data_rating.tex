\doxysubsection{Detailed Description}
Clase que contiene los datos de todos los Rating del sistema. 

Ofrece operaciones para que el Data\+Manager pueda añadir Rating nuevos a partir de un String que sería la representación del Rating a añadir en formato C\+SV. También ofrece operaciones para que el Data\+Manager pida todos los Rating, un Rating en concreto a partir de la posición del Array (útil para bucles) o incluso un Rating a partir de sus I\+Ds.
