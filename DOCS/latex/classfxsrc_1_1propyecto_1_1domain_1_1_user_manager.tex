\doxysubsection{Detailed Description}
Clase singleton encargada de recibir peticiones acerca de usuarios y servirlas. 

Ofrece operaciones para que el usuario pueda hacer Log\+In, Log\+Out, eliminar cuenta y cambiar su perfil, entre otras. Será la clase con la que se comunicará toda la capa de presentación relativa a cuestiones del usuario y esta delegará el trabajo que tenga que ver con los datos almacenados de los usuarios a la clase Data Manager.
